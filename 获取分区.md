## 获取全部分区



### Request URL

> https://app.bilibili.com/x/v2/channel/region/list



### Request Method

> Get





### Request Parameter

| 参数名             | 内容         | 必填/可选 |
| ------------------ | ------------ | --------- |
| appkey             | APP密钥      | 必填      |
| build              | 未知         | 必填      |
| c_locale           | 语言标签     | 必填      |
| channel            | 通道         | 必填      |
| mobi_app           | 手机类型     | 必填      |
| platform           | 平台类型     | 必填      |
| s_locale           | 语言标签     | 必填      |
| statistics         | 未知         | 必填      |
| teenagerqings_mode | 青少年模式？ | 必填      |
| ts                 | 时间戳       | 必填      |
| sign               | 签名加密     | 必填      |
